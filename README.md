# Felena
App web para auditoria de remuneração de servidor

## Sobre o Projeto

Felena é uma aplicação web desenvolvida para realizar auditorias de salários de servidores públicos municipais. A plataforma foi criada para atender a uma necessidade crítica: muitos servidores municipais enfrentam constantemente problemas com seus pagamentos mensais, incluindo atrasos, descontos indevidos e inconsistências nos cálculos de remuneração.

## Problemas Solucionados

- **Atrasos no Pagamento**: Monitora e registra atrasos nos pagamentos mensais
- **Erros de Cálculo**: Valida automaticamente os cálculos de salários e descontos
- **Inconsistências de Dados**: Identifica discrepâncias nos registros de remuneração
- **Falta de Transparência**: Fornece relatórios detalhados e auditáveis dos pagamentos
- **Dificuldade de Rastreamento**: Permite rastreamento histórico completo de todas as transações

## Funcionalidades Principais

- Cadastro e gestão de servidores
- Registro de histórico de pagamentos
- Validação automática de cálculos
- Geração de relatórios auditáveis
- Sistema de alertas para anomalias
- Integração com sistemas municipais
